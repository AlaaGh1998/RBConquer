# RBConquer MMORPG Game

## Overview
RBConquer is an MMORPG game developed using a C# server and a C++ client, utilizing DirectX 11 for rendering. This project provides a comprehensive implementation of game server and client communication, authentication, and game mechanics.

## Setup Instructions
### Server Setup
1. Clone the repository.
2. Open `RBConquer.Server.csproj` in Visual Studio.
3. Restore NuGet packages.
4. Configure the database in `Database/schema.sql`.
5. Run the server.

### Client Setup
1. Clone the repository.
2. Open `RBConquer.Client.sln` in Visual Studio.
3. Setup DirectX SDK and include paths.
4. Build and run the client.
