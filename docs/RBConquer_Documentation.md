# RBConquer MMORPG Documentation

## Setup Guide
### Installation Instructions
1. Download the game client and server files.
2. Extract the files to a suitable directory.
3. Follow the instruction in the Client configuration section to configure your client.

### Server Configuration
1. Ensure the server is running on a compatible operating system.
2. Modify the server settings in the `server.ini` file to match your environment.

### Database Connection Details
- Host: `localhost`
- Database Name: `RBConquer`
- Username: `root`
- Password: `yourpassword`

### Client Configuration
Edit the `client.ini` file to set your server's IP address and port.

### DirectX Setup Requirements
- DirectX Version: DirectX 9.0c or higher
- Install it from the official Microsoft website if not already present.

### Build Instructions
1. Open the project in your preferred IDE (Visual Studio recommended).
2. Make sure all dependencies are installed.
3. Build the project using the appropriate configuration (Debug/Release).

### Compilation Steps
1. Navigate to the solution directory.
2. Run the command `msbuild RBConquer.sln` from the command line.
